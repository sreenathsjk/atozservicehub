# AtoZ Service Hub

## Upload Instructions for GitHub Pages

Upload ALL .html files directly to your repo root. No subfolders needed.

Your repo should look like this:
```
index.html
about.html
community.html
contact.html
daily-updates.html
free-courses.html
govt-schemes.html
latest-jobs.html
privacy.html
service-providers.html
terms.html
README.md
```

Then: Settings → Pages → Source: main branch → / (root) → Save
